Chapter 2c - Connective Tissue

  1. Fluid connective tissue does not include?
    1. erythrocytes
    2. leukocytes
    3. lymphocytes
    4. lymph
  2. Which is not a correct pair of connective tissue category and function?
    1. fluid (transport)
    2. proper (connect and protect)
    3. supporting (structure and strength)
    4. adipose (insulation, shock absorption, enegry)
  3. Which is not a correct pair of connective tissue category and example?
    1. fluid (blood and lymph)
    2. proper (fat, collagen: tendons/ligaments, macrophages)
    3. supporting (cartilage and bone)
    4. adipose (white and brown fat)
  4. Muscle tissue is connective tissue?
    1. true
    2. false
  5. Which is a not a correct cartilege classification?
    1. hyaline cartilege (tough & flexible joints and ribs)
    2. elastic cartilege (external ear)
    3. perichondrium (avascular cartilege layers)
    4. fibrocartilege (tough & limiting joints, pubic bones, vertebrae)
  6. Which is a not correct connective tissue proper, fiber classification?
    1. collagen fiber (tendons/ligaments, resist force in one direction)
    2. elastic fiber (vertebrae ligaments, return to original length)
    3. reticular fiber (organ sheaths, resist force in many directions)
    4. hyaline fiber (ribs, flexible joints)
  7. Adipoctytes in adults can divide?
    1. true
    2. false
  8. Which is a not correct loose connective tissue classification?
    1. areolar tissue (viscous ground substance, deep skin dermis)
    2. adipose tissue (insulation, shock absorption, energy)
    3. reticular tissue (3D stroma that supports organs)
    4. hyaline tissue (extracellular sodium and potassium)
  9. Which is a not correct layer of fascia?
    1. subcutaneous/hypodermis/superficial fascia (between skin and deep fascia)
    2. deep fascia (between subcutaneous fascia and subserous fascia)
    3. subserous fascia (between deep fascia and serous membrane)
    4. visceral fascia (between subcutaneous fascia and subserous fascia)
  10. Which membrane and example pair is incorrect?
    1. muscous membrane (lines passageways with external connections)
    2. serous membrane (lines cavities that do not open to outside)
    3. cutaneous membrane (keratinized, simple squamous epithelium)
    4. synovial membrane (lines synovial joint cavities)
