Chapter 10 - Respiratory System

  1. The upper respiratory system contains all but:
    1. larynx (voice box)
    2. nose
    3. nasal cavity
    4. paranasal sinuses
    5. pharynx (throat)
  2. The lower respiratory system contains all but:
    1. larynx (voice box)
    2. trachea (windpipe)
    3. bronchi and bronchioles
    4. alveoli
    5. pharynx (throat)
  3. The respiratory portion of the respiratory system includes all but:
    1. bronchioles
    2. alveoli
    3. bronchi
  4. Which is part of the conducting portion of the respiratory system?
    1. nose
    2. nasal cavity
    3. bronchioles
    4. alveoli
  5. Place in order from nose to trachea.
    1. nasal cavity → nasopharynx → oropharynx → laryngopharynx → glotis → larynx
    2. nasal cavity → oropharynx → nasopharynx → laryngopharynx → glotis → larynx
    3. nasal cavity → oropharynx → nasopharynx → laryngopharynx → larynx → glotis
    4. nasal cavity → nasopharynx → oropharynx → laryngopharynx → larynx → glotis
  6. The left lung has __________ lobes, and the right lung has __________ lobes.
    1. 2, 3
    2. 3, 2
    3. 2, 2
    4. 3, 3
  7. Place air passage in order from inhalation to gas exchange.
    1. primary bronchus → lobar bronchus → bronchioles → terminal bronchiole → respiratory bronchiole → alveolar ducts → alveoli
    2. primary bronchus → lobar bronchus → bronchioles → respiratory bronchiole → terminal bronchiole → alveolar ducts → alveoli
    3. primary bronchus → bronchioles → lobar bronchus → terminal bronchiole → respiratory bronchiole → alveolar ducts → alveoli
    4. primary bronchus → lobar bronchus → bronchioles → terminal bronchiole → respiratory bronchiole → alveoli → alveolar ducts
  8. Air outside the lung but within the pleural cavity is called __________, resulting in __________ (collapsed lung).
    1. pneumothorax, atelectasis
    2. atelectasis, pneumothorax
    3. respiratory distress, pneumothorax
    4. atelectasis, respiratory distress
  9. The mucociliary escalator moves mucus down the respiratory system.
    1. true
    2. false
  10. The opening of the larynx is the __________.
    1. glottis
    2. epiglottis
    3. trachea
    4. pharynx
